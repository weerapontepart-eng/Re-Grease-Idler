ตั้งค่าการเชื่อมต่อ Google Sheet

เชื่อมต่อแอปกับ Google Sheet ของคุณเพื่อบันทึก/อ่านข้อมูลออนไลน์

การเชื่อมต่อ
Sheet ID ปัจจุบัน: — ยังไม่ตั้งค่า —
Script URL ปัจจุบัน: — ยังไม่ตั้งค่า —
วิธีตั้งค่า (ครั้งเดียว)
  1. สร้าง Google Sheet ใหม่ → คัดลอก Sheet ID จาก URL (ส่วนระหว่าง /d/ กับ /edit) มาวางในช่องด้านบน
  2. ในชีตนั้น เปิดเมนู Extensions → Apps Script
  3. ลบโค้ดเดิมทั้งหมด แล้ววางโค้ดด้านล่างนี้ → กด Save
  4. กด Deploy → New deployment → Type: Web app
    ตั้ง Execute as: Me และ Who has access: Anyone → Deploy → อนุญาตสิทธิ์
  5. คัดลอก Web app URL (ลงท้าย /exec) มาวางในช่อง Apps Script URL ด้านบน
  6. เปิดสิทธิ์การอ่านชีต: Share → General access → Anyone with the link → Viewer
  7. กด บันทึกการตั้งค่า แล้ว ทดสอบการเชื่อมต่อ
